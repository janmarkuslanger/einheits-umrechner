# Einheits-Umrechner

Einfache statische Seite, die Gramm und Milliliter für typische Küchenzutaten umrechnet. Die Seite lässt sich direkt über GitHub Pages hosten.

## Entwicklung
Öffne einfach `index.html` im Browser oder hoste das Repository mit GitHub Pages. Die Logik liegt in `index.js` und nutzt Dichtewerte, um Gramm in Milliliter (und umgekehrt) zu berechnen.
